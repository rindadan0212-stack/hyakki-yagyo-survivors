# 百鬼夜行サバイバーズ — HYAKKI YAGYO SURVIVORS

Vampire Survivors ライクな和風サバイバーアクション。
**統一パレットのドット絵** — 主人公は専用ラスタースプライト、妖・ボス・武器・小物は Canvas 手続き描画を低解像度へ再レンダリング。音は WebAudio シンセ BGM+SFX。
依存ライブラリなし・ビルド不要・ブラウザだけで動く。

## 遊び方

**起動**: `起動.bat` をダブルクリック (または `index.html` を直接ブラウザで開く)

| 操作 | キー |
|---|---|
| 移動 | WASD / 矢印キー (タッチ: 画面ドラッグ) |
| **照準** | **マウスカーソル** (残月・太刀・鎌鼬などの指向性攻撃が刺さる。4 秒動かさなければ移動方向に戻る) |
| 攻撃 | 自動 |
| **奥義** | `Space` (討伐でゲージが満ちる「ここぞ」の一手。出陣支度で 4 種から 1 つ編成) |
| **技** | `Shift` / `E` (クールタイム式。出陣支度で 4 種から 1 つ編成、鍛錬で 3 段強化) |
| レベルアップ選択 | クリック / `1-4` / `←→`+`Enter`、`R` 引き直し、`Esc` 見送り |
| 一時停止 | `Esc` / `P` |
| 消音 | `M` |
| リトライ (結果画面) | `R` |

**目的**: 黄昏から夜明けまで生き延びる。妖を倒して魂を集め、レベルアップで武器と宝具を重ねる。
**決戦**: 夜明けの刻にボスが生きていると夜は明けない――雑魚は焼失し、空が深紅に染まる「決戦」へ突入。ボスを討ち果たすまで続く。

### 夜契り (一夜に 3 回訪れるリスク選択)

夜の進行に応じて、二つの試練と安全な祈りから一つを選ぶ。試練中は敵の速度・攻撃・出現量などが一時的に強化され、時間内に討伐・連撃・強敵討伐・魂回収・無傷生存の目標を果たすと、その夜だけ残る強力な「契印」を得る。失敗時に恒久的な罰はない。

- **修羅 / 連祓 / 首狩 / 魂喰 / 無傷**: ビルドと現在の戦況に合わせて危難を選び、威力・発動・会心・範囲・防御などを伸ばす
- **静謐の祈り**: 試練を退き、体力と奥義を回復する
- 操作はクリック、または `1-3`

レベルアップだけに成長判断を集中させず、「今のビルドでどこまで危険を引き受けるか」という中盤の目標と緩急を作るためのシステム。

### 秘術 (レベルアップで最大4系統)

通常の武器・宝具強化とは別に、発動条件そのものを増やすラン内技能。各3段まで強化できる。

| 秘術 | 戦い方の変化 |
|---|---|
| 魂爆の印 | 一定数を討つたび、止めを刺した地点から連鎖爆発 |
| 残照の型 | 提灯の灯内で威力と発動速度が上がる |
| 夜渡り | 闇の中で移動が速まり、連撃の猶予が伸びる |
| 火守の誓い | 灯内の防御、提灯の耐久、灯勢の獲得量を強化 |
| 魂寄せの環 | 魂を一定数拾うたび周囲を吸引し、技と結界の合間を短縮 |
| 空蝉の法 | 一定時間ごとに被弾を一度無効化して反撃 |
| 火継ぎの作法 | 提灯の点灯時に体力・奥義・灯勢を得る |

### 三灯システム (提灯の戦術拠点化)

提灯は近づくだけでは点かず、圏内に短時間留まって火を結ぶ。灯内で妖を倒すと灯勢が蓄積し、満ちると灯紋固有の大技が発動する。

- **紅灯**: 妖への打撃と鈍化が強く、満ちると広範囲を焼き払う
- **蒼灯**: 発動と魂吸引を強化し、満ちると魂を呼び技・結界を早める
- **白灯**: 回復と防御を与え、満ちると敵弾を消して金剛を授ける
- 初点灯3基ごとに**三灯共鳴**が発動。画面の妖を祓い、回復と荒魂を得る
- **油赤子**は灯勢の高い提灯を優先して消す。消灯後はすぐに点け直せない

### 奉納 ― 絵馬掛け (永続強化)

タイトルの「奉納」から、小判を納めて**全ランに効く永続の加護**を授かる。
9 種の絵馬 (力・体・足・刻・縁・守・才・銭・福) に各 3〜5 段。段が上がるほど高額 (総額 16,000 両超)。負けた夜の小判も無駄にならない。

### 鍛錬 ― 使い手と得物の打ち直し (永続強化)

タイトルの「鍛錬」から、小判で**使い手 (各 5 段) と武器 (各 3 段)** を恒久強化できる。

- **使い手**: 段ごとにキャラ固有の伸び (晴=威力/発動、鈴=範囲/吸引、無月=威力/会心)。**三段で特殊能力が目覚める**:
  - 晴 「森羅の目」 ― レベルアップの選択肢が 4 つになる
  - 鈴 「祓いの舞」 ― 体力が 3 割を切ると自動で大祓+回復 (60 秒に一度)
  - 無月 「残心」 ― 会心で妖を斬り伏せた時、月波が周囲を薙ぐ
- **得物**: 全 18 武器それぞれ、段ごとに威力 +4% (最大 +12%)

## ゲーム内容

- **武器 12 種** (最大 6 つ、Lv7 まで): 破魔の御札 / 太刀「燕」 / 式神・白狐 / 雷霆符 / 狐火 / 鎌鼬の風 / 浄化の結界 / 大数珠 / 梵鐘 / 破魔矢 / 狛犬 / 御幣
- **宝具 12 種** (最大 6 つ、Lv5 まで): 力の勾玉 / 天狗の羽団扇 / 韋駄天の足袋 / 達磨の御守 / 霊泉の雫 / 神楽鈴 / 磁鉄のお守り / 鉄の鉢金 / 朱の御印 / 貫きの鏃 (弾の貫通 +1) / 跳ね鞠 (弾が妖から妖へ跳弾) / 澄心の弓懸 (足を止めて 2 秒で連射 最大 -40%)
  - **流派の競合は抽選に出ない**: 貫きの鏃⇔跳ね鞠は相互排他。手裏剣 (跳弾武器) 所持中は貫きの鏃が出ず、貫きの鏃所持中は手裏剣が出ない
  - **死にスキルは抽選に出ない**: 貫きの鏃・跳ね鞠は「貫通の効く弾武器」(御札/狐火/破魔矢/御幣/手裏剣) を持っている時だけ候補になる
  - 跳ね鞠は手裏剣の固有跳弾と**加算**で重複する (手裏剣 Lv1 跳弾2 + 跳ね鞠 Lv5 = 跳弾7)
- **レベルアップの選択操作**: **引き直し ×2 / 封印 ×2 / 見送り** が毎ラン使える。封印した札はそのランの抽選 (宝箱含む) から消える
- **覚醒**: 武器 Lv7 + 対応する宝具 Lv3 で武器が覚醒し、大幅強化。**カードに覚醒の道筋が常に表示される**
- **限界突破**: 武器・宝具を全て極めた後は、レベルアップで武器を上限の先へ (1 回ごと威力 +8%、際限なし)
- **実績**: プレイ内容で武器・キャラ・小判が解放される (小判購入と並ぶもう一つの道)。一覧はタイトルの「絵巻」
- **絵巻 (図鑑)**: 実績 / 討った妖の図鑑 (未討伐は影) / 全武器の覚醒ペア一覧 / 宝具の共鳴・排他関係
- **百鬼語り**: 全 25 妖に**伝承由来の語り** (討って初めて絵巻に記される)。死亡画面には**殺した妖ごとの固有の一文** (夜明け直前・決戦中は特別な句)、夜明けには**ステージ×使い手ごとの結び**
- **エリートの二つ名**: 出現時に「"夜啼き" の赤鬼」と名乗り、異名ごとに性質が変わる — 韋駄天 (俊足) / 巌 (堅牢) / 怨念 (死亡時に弾を撒く) / 福持ち (加護 2 個) / 銭袋 (小判 3 倍) / 手負い (脆いが速い) / 魂満ち (経験 2 倍) など全 10 種
- **共鳴**: 特定の宝具 2 種を揃えると自動発動するセット効果。完成する取得カードには「◆ 共鳴」予告が出る
  - **穿心** (鏃×御印): 弾が妖を穿つたび威力 +12% ずつ加速
  - **鞠唄** (鞠×磁鉄): 跳弾 +1、跳ね移れる間合い 240→430
  - **神楽** (足袋×鈴): 駆けている間、発動 -12%
  - **不動** (御守×鉢金): 被弾の刹那、周囲へ反撃ノヴァ + 弾き飛ばし
- **死体物理**: 倒した妖は**止めの一撃の方向と威力で吹き飛び**、回転しながら滑って**他の妖を撥ね飛ばす** (連鎖キルあり)。爆発は死体を放射状に飛ばし、灯篭は破片が弾ける
- **連撃コンボ**: 撃破を途切れさせないとコンボが伸び、節目で魂ボーナス。100 連超えで体が燃え始める
- **加護ドロップ**: 妖が稀に落とす光球で一時強化 — 荒魂 (威力2倍) / 神速 (移動+連射) / 金剛 (無敵) / 分身 (全武器の弾+1)。エリートは高確率で落とす
- **宝匣 (宝箱)**: マップ遠方に番人付きの唐櫃が出現。**方角と距離 (間) はプレイヤーの周囲に常時表示**。開封で**おにぎり + 魂が確定**、さらに候補 (1個 60% / 2個 39% / 3個=大当タリ 1%) から**一つだけ選んで**取得 — 要らなければ追加のおにぎりを選べる (強制取得なし)
- **妖 20 種** (全種専用2フレームドット絵、ステージごとに顔ぶれが変わる) + **色分けエリート** + **ボス 6 種** ― 化け狸 / 濡女 / 鵺 / 牛鬼 / がしゃどくろ / 酒呑童子。**全ボスが 3 つ以上の攻撃パターン**を持つ (突進・衝撃波・弾幕・落雷・召喚など。危険な技には予兆あり)
- **イベントウェーブ**: 蝙蝠の環、夜烏の嵐、骸骨の囲い など
- **三色提灯**を灯して育て、油赤子から防衛する。点灯中の圏内はビルドに応じた攻め所・守り所になる
- ハイスコアは localStorage に保存

## 技術メモ

| 項目 | 実装 |
|---|---|
| 描画 | Canvas 2D。専用キャラクターPNG + 手続き生成した妖・ボスを共通パレットのドット絵へ変換しオフスクリーンキャッシュ |
| 照明 | 夜の暗幕キャンバスに destination-out で光を抜く 2 パス合成 |
| ループ | 固定タイムステップ 60Hz (タイムスケール/ヒットストップ対応) |
| 衝突 | 空間ハッシュグリッド (80px セル)。敵 430 体 + 弾幕で 60fps |
| 音 | WebAudio 全合成。A 平調子で**作曲された 32 小節のテーマ曲** (主題×2 → サビ → 終結)。琴・琵琶は Karplus-Strong 弦振動の事前レンダリング、笛は遅延ビブラート、囃子は大太鼓/締太鼓/縁打ち/拍子木/神楽鈴。夜の深度・低体力・100連撃・夜契り・ボス戦に反応して楽器層と強度が変化。秘術、魂爆、魂寄せ、提灯の点灯・消灯・灯勢解放にも専用SFXを持ち、大音時はBGMを自動ダッキング |
| UI | HUD は Canvas、メニューは DOM。朱・金・月光色で役割を分けた和風16bit UI |
| 敵素材 | 20妖+6ボスを個別ラスタードット絵化。各2フレーム、既存の当たり判定・足元アンカーを維持 |

### デバッグ API (DevTools コンソール)

```js
__G.stats()        // 現在の状態スナップショット
__G.ts(8)          // タイムスケール 8 倍
__G.auto()         // オートパイロット
__G.autoLevel()    // レベルアップ自動選択
__G.god()          // 無敵
__G.show()         // fps/エンティティ数オーバーレイ (?debug でも可)
```
